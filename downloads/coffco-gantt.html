CoffCo Hotel Coffee Machine

Build, Integration & Verification Schedule · 2027

Ref Activity Deliverable / Scope Lead Subsystem Effort Category
P-01 Power Supply Unit — COTS procurement Identify and place order for certified universal-input PSU module (110–240V AC). Qualify to IEC 60335. Receive, inspect, and bench-verify against EI-E-01 spec. SS-4 (Power) 3M Procurement
P-02 Structural chassis — tooling and moulding Release injection mould tooling for chassis and casing to CoffCo design. Trial shots, dimensional check, and approval before integration begins. Long lead — ordered at programme start. SS-6 (Chassis) 4M Procurement
B-01 Control & Monitoring Assembly — build PCBA manufacture, component placement, and reflow. Load firmware baseline. Bench smoke-test: power rails, MCU boot, basic I/O continuity. SS-2 / SS-5 2M Subsystem Build
B-02 Thermal Assembly — build and bench-leak Assemble heating element, pump, Peltier cooler, switching valve, and dispensing nozzle into the thermal sub-assembly. Pressure-test fluid path at 0.5 bar. Food-safety material certificates on file. SS-2/3 (Thermal) 3M Subsystem Build
B-03 Fluid Store — reservoir, cassette, sensors Assemble water reservoir, consumable cassette housing, drip tray, and level-sensing elements. Confirm tool-free access panel operation. Leak check at atmospheric pressure. SS-3 (Fluid) 2M Subsystem Build
B-04 HMI / User Interface Panel — build & UX iteration Build physical fascia, rotary volume dial, coffee and milk buttons, and LED status ring. Run two UX iteration rounds against SRS 2.1 (unaided first-time use). Longer build cycle than other subsystems — hardware and firmware developed in parallel; UX prototype must pass before panel is frozen. SS-1 (HMI) 5M Subsystem Build
V-01 PSU stand-alone electrical check Dual-voltage test across UK / EU / US plug modules. Load regulation under simulated peak draw. Isolation check — no accessible live surface. EMC pre-scan. Gateway to firmware integration. SS-4 (Power) 2W Verification
S-01 Firmware integration — state machine & protocol stack Bring together MCU firmware modules: 8-state FSM, brew sequencer, I2C protocol stack for all II-D interfaces, sensor polling loops, and non-volatile fault logger. Tested against hardware stubs and simulated sensor harness — no real thermal or fluid load yet. SS-2 / SS-5 6W SW / FW Integration
S-02 Wireless telemetry integration (TD-06 — OA-01) Integrate wireless module once technology confirmed (TD-06 closure is a prerequisite). Commission MQTT-over-TLS telemetry payload: machine ID, FSM state, consumable level, fault code. Requires hotel IT infrastructure confirmation from Vacation Stables. SS-5 (Comms) 3W SW / FW Integration
V-02 Control Assembly bench verification Verify all II-D interfaces at stub level: I2C brew command / ACK, FSM state transitions, dry-fire interlock (SRS 1.10), low-consumable threshold, telemetry payload receipt at cloud test endpoint. All C2.x ICD test steps must pass before hardware integration begins. SS-2 / SS-5 2W Verification
I-01 Thermal integration — Control + PSU + Thermal stack Bring Thermal Assembly onto Control + PSU bench stack. Close-loop brew command → temperature and pressure feedback (II-D-03, II-D-05). Validate extraction at 0.9–1.3 bar, brew cycle ≤180s, hardware thermal cutoff at 105°C, and coffee/milk flush cycle (II-F-01 cross-contamination check). SS-2/3/4 6W HW Integration
V-03 Thermal stack verification T3.1–T3.7 ICD steps: temperature accuracy, pressure range, 180s cycle timing, cutoff trip test, contamination rinse, dispensing clearance ≥105mm (SRS 2.7), americano ≥60°C and milk 60–90°C at outlet. SS-2/3/4 2W Verification
I-02 Fluid Store integration — level sensing and safety interlock Add Fluid Store to verified thermal + control stack. Commission water coupling (II-M-01), drip return path (II-M-02), and level sensor data to Control Assembly (II-D-04). Confirm consumable sufficiency gate (SRS 1.10 / C-12) blocks brew-start correctly. SS-3/4 4W HW Integration
V-04 Fluid system verification F4.1–F4.5 ICD steps: 10 consecutive brews without refill, dry-fire block active at threshold, drip tray overflow alert, housekeeping restock cycle under 5 minutes with no tools, fluid pathway leak inspection after 10 brews. SS-3/4 2W Verification
I-03 HMI integration — UI panel onto full stack Connect built-and-iterated HMI panel to Control Assembly. Confirm I2C brew command handshake (II-D-01), status LED drive (II-D-02), parallel coffee-and-milk brew paths, and volume dial accuracy across 200–400ml range. SS-1/2 4W HW Integration
V-05 HMI and unaided user trial U5.1–U5.5 ICD steps: button / dial functional coverage, all four LED states confirmed, parallel brew path simultaneous activation, volume within ±5ml, unaided first-use trial by test subject with no prior briefing — target ≥4/5 usability score (SRS 2.1). SS-1/2 2W Verification
I-04 Full system assembly — all subsystems into chassis Mount Control Assembly, Thermal Assembly, Fluid Store, PSU, and HMI Panel into the Structural Chassis. Route and dress internal harnesses. Close housekeeping access panel. Fit anti-theft anchor lug. Confirm overall envelope and mass within SRS 3.2 and 4.10. SS-6 (all) 3W HW Integration
V-06 System-level and compliance verification S6.1–S6.8 ICD steps: 300×300×300mm envelope, ≤10kg mass, anti-theft lug 500N pull, ≥80% black/transparent surface, full brew cycle ≤180s in-chassis, EMC to EN 55014 Class B, IEC 60335 / CE / UKCA safety compliance, 36-hour endurance run at 2 brews/hour (SRS 3.3 acceleration proxy). All SS 3W Verification
FAT Factory Acceptance Test — Vacation Stables witness Formal acceptance event against full COFFCO-SRS-001 requirement set (all 38 requirements). Vacation Stables engineering and operations representatives as witnesses. Signed FAT report is the exit criterion for the integration programme. CoffCo / VS 1M Acceptance
