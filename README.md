## Interactive Features

- **HMI Prototype** — Full machine UI simulation with all 7 states, two-step brew confirmation, real-time telemetry log
- **SRS Explorer** — Filterable requirements table, click to expand rationale and acceptance criteria
- **Safety Risk Matrix** — MIL-STD-882E interactive risk matrix, click hazards to see full HAZID entries
- **FIDA Fault Simulator** — Walk through the three primary maintenance procedures interactively
- **Architecture Diagrams** — Full interactive architecture HTML with SVG diagrams

## Deployment

This site deploys automatically to GitHub Pages via GitHub Actions on every push to `main`.

To deploy manually:
1. Fork or clone this repository
2. Enable GitHub Pages in repository Settings → Pages → Source: GitHub Actions
3. Push to `main` — site deploys automatically
